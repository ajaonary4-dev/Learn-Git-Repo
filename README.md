# LEARN GIT
This file is used to learn how to use the Terminal/Command Line and use basic text commands.
It is used to practice using GIT: Create files, update, revert to past code versions or create multiple versions of the same code.
Learn how git is used to collaborate within teams.
I will learn how to use commands such as: git init, git add, git commit...

## How to clone repo to local computer and use first git commands

1 - git clone <url>
2 - cd <filename>
3 - edit file 
4 - check git status - git status - modified
5 - Create new file - git status - untracked
6 - add files to git so they can be tracked
6a - git add <file> - adds new file(untracked)
6b - git add . - adds all files to track(modified and untracked)
7 - git commit -m - saves changes on local computer only -First -m is title and Second -m is short description
